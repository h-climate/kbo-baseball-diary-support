응원노트

지원 페이지

응원노트는 공식 KBO, 구단, 포털, 영상 플랫폼 앱이 아닌 개인 팬 기록장입니다.

문의하기

앱 사용 중 문제가 있거나 개인정보 관련 문의가 있으면 아래 공개 이슈 페이지에 남겨 주세요. 공개적으로 남기기 어려운 개인정보, 민감한 정보, 첨부 파일 원본은 이슈에 올리지 마세요.

GitHub Issues에서 문의하기

앱에서 저장하는 내용

  • 선택한 응원팀
  • 일기 제목, 본문, 기분, 직관 여부
  • 사용자가 직접 선택한 사진과 파일 첨부
  • 사용자가 직접 입력한 경기 기록 보정 정보

현재 제출 범위에서는 계정 생성, 서버 동기화, 광고, 구매, 분석 SDK, 푸시 알림, YouTube 하이라이트 연결, 자동 하이라이트 매칭을 제공하지 않습니다.

기본 문제 해결

  • 첨부 파일이 보이지 않으면 파일이 앱 안으로 복사되었는지 다시 저장해 주세요.
  • 경기 데이터가 기대와 다르면 일기 화면에서 직접 기록을 보정해 주세요.
  • 앱을 삭제하면 기기에 저장된 일기와 첨부 파일도 함께 삭제될 수 있습니다.
