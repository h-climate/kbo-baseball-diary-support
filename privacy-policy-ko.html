응원노트

개인정보 처리방침

시행일: 2026-06-28

요약

응원노트는 야구 팬이 자신의 기기에 일기와 첨부 파일을 저장하는 개인 기록장입니다. 현재 제출 범위에서는 계정, 광고, 구매, 분석 SDK, 서버 동기화, 추적 기능을 제공하지 않으며, 앱에서 작성한 일기와 사용자가 선택한 첨부 파일은 기기 안에 저장됩니다.

수집 또는 저장되는 정보

앱은 다음 정보를 사용자의 기기에 저장할 수 있습니다.

  • 선택한 응원팀
  • 일기 제목, 본문, 기분, 직관 여부
  • 사용자가 직접 선택한 사진과 파일 첨부
  • 사용자가 직접 입력한 경기 기록 보정 정보
  • 앱 사용에 필요한 로컬 설정과 캐시

사진과 파일 접근

응원노트는 사용자가 직접 선택한 사진 또는 파일만 앱 저장 공간으로 복사합니다. 전체 사진 보관함을 스캔하거나, 사용자가 선택하지 않은 파일을 가져오지 않습니다.

네트워크와 제3자 데이터

현재 공개 제출 빌드는 비공식 자동 경기 데이터 수집을 기본으로 실행하지 않습니다. 실험용 빌드에서 외부 경기 데이터 확인 기능이 명시적으로 켜진 경우에도, 앱은 선택한 경기 맥락을 확인하기 위한 요청만 수행하며 사용자의 일기 본문이나 첨부 파일을 외부로 전송하지 않습니다.

추적, 광고, 분석

응원노트는 사용자를 추적하지 않으며, 광고 SDK나 분석 SDK를 포함하지 않습니다. 다른 회사의 앱이나 웹사이트를 넘나드는 추적 목적으로 데이터를 사용하지 않습니다.

계정과 서버 동기화

현재 제출 범위에서는 계정 생성, 로그인, 서버 동기화 기능을 제공하지 않습니다. 일기와 첨부 파일은 사용자의 기기에 저장됩니다.

삭제

앱 안에서 일기를 삭제하면 해당 일기 기록과 연결된 로컬 첨부 정보가 삭제됩니다. 앱을 삭제하면 기기에 저장된 응원노트 데이터도 함께 삭제될 수 있습니다.

아동 개인정보

응원노트는 아동으로부터 개인정보를 의도적으로 수집하지 않습니다.

변경

개인정보 처리방침이 변경되면 이 페이지의 시행일을 갱신합니다. 서버 동기화, 계정, 분석, 광고 등 개인정보 처리 범위가 커지는 기능을 추가하는 경우 App Store 메타데이터와 이 방침을 함께 업데이트합니다.

문의

개인정보 또는 앱 지원 문의는 응원노트 지원 이슈 페이지를 사용해 주세요. 공개적으로 남기기 어려운 개인정보나 민감한 정보는 이슈에 올리지 마세요.
